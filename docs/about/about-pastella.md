# About Pastella

Pastella is a next-generation cryptocurrency combining **RandomX Proof-of-Work** with **Simple Reward Staking** and **on-chain governance**.

## What Makes Pastella Different?

Pastella isn't just another cryptocurrency—it's a complete ecosystem designed for long-term sustainability and decentralized governance.

### Core Features

| Feature | Description |
|---------|-------------|
| **Hybrid Consensus** | Combines RandomX PoW with Simple Reward Staking for network security |
| **On-Chain Governance** | Decentralized proposal and voting system for protocol upgrades |
| **Multi-Tier Staking** | Variable annual rewards (5%-50%) based on lock duration |
| **Transparent Ledger** | All transactions publicly visible on the blockchain |
| **Democratic Upgrades** | 90% voting threshold for protocol changes |

## Governance

Pastella features a decentralized governance system where stakeholders can propose and vote on protocol changes. Voting power is determined by both the amount staked and the lock duration—longer stakes get higher voting multipliers.

### Voting Thresholds

- **51%** - Parameter changes (fees, block size)
- **67%** - Protocol upgrades
- **75%** - Treasury spending and consensus changes
