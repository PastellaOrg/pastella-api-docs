# Contributors

Pastella is made possible by a diverse community of developers, researchers, and enthusiasts from around the world.

## Core Team

The Pastella project is maintained by the core development team who contribute to:

- **PastellaCore** - Core protocol implementation
- **Mining pool software** - Infrastructure for miners
- **Block explorer** - Network transparency tools
- **Wallet applications** - Desktop and mobile wallets
- **Developer utilities** - Libraries and tools

## Recognition

We appreciate every contribution, whether it's:

- Writing code
- Testing and bug reporting
- Documentation
- Community support
- Translation
- Design and UX

## Notable Contributors

We'd like to thank all contributors who have helped make Pastella what it is today. Your time, expertise, and passion drive this project forward.

## Become a Contributor

Want to join? Start by:

1. Joining our [Discord](https://discord.gg/9jqwc4UWrK)
2. Checking the [Contributing Guide](contributing)
3. Finding a good first issue to tackle
4. Reading the [Developer Guide](../developer/help-support)

Every contribution counts, no matter how small!
