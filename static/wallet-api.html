Pastella Wallet API (1.0.0)

Download OpenAPI specification:

License: MIT

Complete REST API documentation for the Pastella cryptocurrency wallet.

Base URL

Default API port: 21002

Authentication

All requests require an API key header:

X-API-KEY: your_rpc_password

Transaction Types

  • TRANSFER: Regular transfer transaction
  • STAKING: Staking transaction

Balance Types

  • unlocked: Spendable balance (confirmed)
  • locked: Unconfirmed/locked balance

Address Format

Pastella addresses start with PAS1 prefix.

Wallet

Wallet management operations

Open an existing wallet

Opens an existing wallet file and connects to the daemon.

Request Body schema: application/json
required
filename
required
string

Path to wallet file

password
required
string

Wallet password

daemonHost
string
Default: "127.0.0.1"

Daemon host address

daemonPort
integer
Default: 21001

Daemon RPC port

daemonSSL
boolean
Default: false

Use SSL for daemon connection

Responses

Local development server

http://localhost:21002/wallet/open

Request samples

  • Payload
  • curl
Content type
application/json
{
  • "filename": "/path/to/wallet.wallet",
  • "password": "secure_password",
  • "daemonHost": "127.0.0.1",
  • "daemonPort": 21001
}

Response samples

  • 200
Content type
application/json
{
  • "status": "OK",
  • "message": "Wallet opened successfully"
}
